# Praxis Seguridad - Sitio Web Corporativo

## Descripción
Sitio web corporativo para **Praxis Seguridad**, consultoría estratégica en seguridad privada con sede en Santomera, Murcia (España).

## Enfoque
- Consultoría estratégica en seguridad privada
- Auditorías y análisis de riesgos
- Diseño e integración de sistemas de seguridad
- Servicios auxiliares y vigilancia
- Automatización e IA aplicada

## Stack Técnico
- **PHP 8+** - Arquitectura modular con includes
- **Tailwind CSS** - Via CDN
- **Google Fonts** - Montserrat + Inter
- **Font Awesome 6** - Iconografía profesional

## Estructura de Archivos
```
├── index.php              # Página principal
├── servicios.php          # Servicios detallados
├── contacto.php           # Contacto y formulario
├── includes/
│   ├── header.php         # Cabecera y navegación
│   └── footer.php         # Pie de página
├── images/                # Imágenes del sitio
├── .htaccess              # Configuración Apache
└── README.md              # Este archivo
```
